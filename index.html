Usar no iPhone sem computador

Para o iPhone guardar os dados sozinho, publique estes arquivos em uma hospedagem estática grátis com HTTPS, abra o link pelo Safari e escolha Adicionar à Tela de Início.

  1. Use uma opção gratuita como GitHub Pages, Cloudflare Pages ou Netlify para hospedar a pasta do sistema.
  2. No Safari do iPhone, abra o endereço HTTPS publicado, como https://seu-usuario.github.io/planeta-locacoes/.
  3. Toque em Compartilhar e depois em Adicionar à Tela de Início.
  4. Abra pelo ícone Planeta Locações. Os cadastros ficam salvos neste iPhone.

O servidor local e o endereço 127.0.0.1 servem apenas para testes no computador. Se já houver dados no computador, exporte um backup aqui e importe no iPhone depois da instalação.
